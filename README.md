# MoreMaths

Python 自建数学模块，补充 `math` 标准库之外的功能。

## 功能

### 数学计算
- 阶乘、斐波那契数列
- 排列数 / 组合数
- 最大公因数（辗转相除）/ 最小公倍数
- 质数检测、分解质因数、底数分解
- 完全平方数判断、二次根式化简、自幂数判断
- 平面直角坐标 ↔ 极坐标互转
- 海伦公式 / 秦九韶公式（三角形面积）
- 幂运算 / 重幂 / 高德纳箭头

### 数据统计
- 算术平均数、加权平均数、中位数、众数
- 方差（总体/样本）、标准差（总体/样本）
- 极差、累乘、平均绝对偏差、异常数检测

### 布尔逻辑
- 与 / 或 / 非 / 与非 / 或非 / 异或 / 同或
- 半加器 / 全加器
- 交通信号灯逻辑

## 安装

```bash
pip install .
```

## 快速示例

```python
from MoreMaths import *

# 阶乘
print(Factorial(5))  # 120

# 组合数
print(Combination(15, 12))  # 455

# 质数检测
print(ifPrime(17))  # True

# 数据统计
data = DataStat(120, 30, 90, 60, 60)
print(data.Arithmetic_mean())  # 72.0

# 布尔逻辑
print(Bool_Sum.XOR(1, 0))  # 1
```
